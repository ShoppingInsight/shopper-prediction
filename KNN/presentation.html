거리 계산 — 유클리드 / 맨해튼

d = √[ (a₁ − b₁)² + (a₂ − b₂)² + ⋯ + (aₙ − bₙ)² ]

본 분석은 13차원 피처 공간에서 Manhattan 거리(p=1)를 사용 — CV 결과로 선정.

왜 회귀를 골랐나 — Classifier의 predict_proba도 결국 이웃 비율이라 수학적으로 동치. 다만 회귀로 풀면 출력이 처음부터 연속 확률 → Brier Score 같은 확률 보정 지표를 자연스럽게 평가 가능.

문제 정의 — "비슷한 사람은 비슷하게 행동한다"

"나랑 비슷하게 둘러본 사람들이 결국 샀다면,
나도 살 가능성이 높지 않을까?"

KNN 선택의 정당성 3가지

  • ① 결과가 "확률"로 자연스럽게 떨어진다 — 이웃 K명 중 구매 비율 → 0~1. 별 가정 없음
  • ② 비즈니스 직관성 — "이 사람과 비슷한 75명 중 80%가 샀다" → 사장님도 이해
  • ③ 비모수적 모델 — 함수 모양("구매 확률 = 0.3 × 체류시간 + …")을 미리 가정하지 않음

K 후보 — 왜 로그 간격인가

이웃 1명이 차지하는 가중치 = 1/K. 따라서 K 변화의 의미는 1/K 변화에 비례.

K=51 → 530.020 → 0.019 (−4%)노이즈 수준

최종 K 후보 (로그 간격 12점)

K_SWEEP = [1, 3, 5, 7, 11, 15, 21, 31, 51, 75, 101, 151]

경험칙 √Ntrain ≈ √9,864 ≈ 99 부근까지 커버 — plateau 확인 가능. 수업 규칙 따라 전부 홀수.

각 선택의 근거

  • K=75 — √Ntrain≈99 경험칙의 근방, plateau 위. 너무 작지도 크지도 않음
  • distance — 가까운 이웃에 더 큰 가중 → 양극화 완화, 노이즈 둔감
  • Manhattan — 우편향 분포 많은 데이터에서 outlier에 강건. Euclidean 제곱 항이 outlier를 부풀리는 걸 회피
인정하는 한계 PDF 23쪽 — K=75가 왜 75인지는 이론이 아닌 실험적 결과일 뿐. 이는 KNN 알고리즘 자체의 한계 ("그냥 이 데이터엔 이게 좋더라").

ROC-AUC — 그게 정확히 뭔가

유래: 2차 세계대전 미군 Receiver Operator(레이더 운용자)가 적기 신호 vs 잡음을 구분하는 능력 측정에서 출발.

계산 — 모든 임계값에서의 분리 능력

  1. 임계값을 1.0 → 0.0으로 천천히 내림
  2. 각 임계값에서 TPR(=Recall) vs FPR 계산
  3. (FPR, TPR) 곡선 아래 면적 = ROC-AUC
가장 깔끔한 해석
"임의의 양성 1개와 음성 1개를 뽑았을 때, 모델이 양성에 더 높은 점수를 줄 확률"
→ 본 분석 0.8830 = 88.3% 확률로 구매자에게 더 높은 점수를 매김. 순위 지표.

한계 — 클래스 불균형이 심하면 낙관적 (음성 99%면 거의 쓸모없는 모델도 0.9 가능). → PR-AUC 보완 필요.

PR-AUC — 불균형 보정용

유래: 정보 검색(Information Retrieval) 분야. "구글 검색 100개 중 진짜 관련 문서는 몇 개?"에서 출발.

왜 둘 다 봤나 — 양성 15.5% 불균형 → ROC-AUC 0.883이 부풀려졌을 위험. PR-AUC 0.619 (베이스라인 4배)로 보정 확인 → 두 지표 모두 양호하므로 성능 신뢰 가능.

Recall이 낮으면 좋은 건가 나쁜 건가?

본 분석 임계값 0.5 기준 — Recall = 0.306 (구매자 100명 중 30명만 잡음, 70명 놓침)

Recall ↑ 좋은 점

  • 양성을 거의 다 잡음
  • 암 진단, 사기 탐지
  • "놓치면 큰일" 상황

Recall ↑ 나쁜 점

  • Precision ↓ (헛양성 증가)
  • 쿠폰 낭비

Recall ↓ 좋은 점

  • Precision ↑ (본 분석: 0.765)
  • 마케팅 예산 효율
  • "확신만 잡자" 보수 운영

Recall ↓ 나쁜 점

  • 기회 손실 큼 (70% 놓침)
임계값 0.5는 한 점일 뿐. KNN이 0~1 점수를 뱉으니 비즈니스에 맞춰 자유 조정 가능. ROC-AUC 0.883이 보장하는 건 임계값 어디서든 순위가 양호하다는 것.
